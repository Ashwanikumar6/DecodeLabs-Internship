## 📌 Project Overview

This project implements an **Iris Flower Classification** model using the **K-Nearest Neighbors (KNN)** algorithm. It demonstrates a complete machine learning workflow including data preprocessing, exploratory data analysis (EDA), feature scaling, hyperparameter tuning, model evaluation, visualization, and model persistence.

Developed as **Project 2** during the **DecodeLabs AI Internship**.

---

## 🎯 Objectives

- Load and analyze the Iris dataset
- Perform Exploratory Data Analysis (EDA)
- Preprocess and standardize the data
- Train a KNN classifier
- Optimize hyperparameters using GridSearchCV
- Evaluate model performance
- Visualize results
- Save the trained model for future predictions
