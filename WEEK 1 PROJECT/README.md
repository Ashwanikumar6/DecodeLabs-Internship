# 🤖 Rule-Based Chatbot

> A simple Rule-Based Chatbot built using Python as part of the **Decode Labs AI Internship – Week 1 Project**.

This project demonstrates the fundamentals of Python programming by creating a chatbot that interacts with users through predefined responses. It also provides useful utilities such as displaying the current time, date, and day while handling user input in a clean and user-friendly manner.

---

## 📌 Features

- 👋 Greets users with random welcome messages
- 💬 Responds to predefined questions
- 🤖 Introduces itself and explains its purpose
- 🧠 Answers basic AI and Python-related questions
- 🕒 Displays the current time
- 📅 Displays today's date
- 📆 Displays the current day
- 🆘 Provides a help menu with supported commands
- 🚪 Supports exit commands (`exit`, `quit`, `bye`)
- 🧹 Cleans user input for better matching
- ❓ Handles unknown questions gracefully
